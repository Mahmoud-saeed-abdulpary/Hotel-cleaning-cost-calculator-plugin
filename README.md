## üìä DATABASE STRUCTURE

The plugin creates 3 custom tables:

### 1. wp_hcc_quotes
**Purpose:** Store all quote requests
**Key Fields:**
- quote_number (unique identifier)
- client information (name, email, phone, address)
- rooms_data (JSON)
- pricing details
- status (pending/approved/rejected)
- timestamps

### 2. wp_hcc_discount_rules
**Purpose:** Store conditional discount rules
**Key Fields:**
- rule name and description
- discount type and value
- conditions (JSON)
- priority
- active status
- date ranges

### 3. wp_hcc_activity_log
**Purpose:** Track all system activities
**Key Fields:**
- action type
- user ID
- object details
- metadata (JSON)
- IP address
- timestamp

**Note:** All tables use proper indexes for performance
